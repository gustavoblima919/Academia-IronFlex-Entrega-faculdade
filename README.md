# Academia-IronFlex-Entrega-faculdade

Banco de dados Academia Iron Flex

Projeto de disciplina de Banco de dados, baseado no minimundo da Academia IronFlex (Fícticio)

 O repositório contém:
- Scripts de criação das tabelas
- Inserts iniciais para dar vida ao banco
- Consultas SELECT para testar a estrutura
- comandos de UPDATE E DELETE
- Instruções sobre como executar

Como executar:

1 Abra o Workbench ou PostgreSQL
2 Execute primeiro o arquivo `crate_table_Sql`.
3 Depois rode `insert_data.sql`para popular o banco.
4 use os arquivos `selects.sql`, `updates.sql` e `delete.sql` para testar.
5 ajuste as queries conforme necessidade.
